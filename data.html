Baby Name Analysis

Language: Python

This project uses data taken from the U.S. Social Security Baby Name Catalog which reports the names given to babies between the years 1880 and 2021. It analyzes components such as individual name frequency, the most popular baby names based on gender, the most popular gender neutral baby names, etc.
